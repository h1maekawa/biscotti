# Biscotti ホームページ

Biscotti様のホームページです。

## 概要

唯一無二のお菓子を作るBiscotti様の公式ホームページです。オーダーメイドのお菓子作りを中心に、店舗情報やご予約方法などをご案内しています。

## 特徴

- モダンで可愛らしいデザイン
- レスポンシブ対応（スマートフォン、タブレット、PC）
- スムーススクロールナビゲーション
- アニメーション効果

## ファイル構成

```
biscotti/
├── index.html      # メインHTMLファイル
├── styles.css      # スタイルシート
├── script.js       # JavaScriptファイル
└── README.md       # このファイル
```

## セクション構成

1. **トップ（ヒーローセクション）**
   - メインビジュアルとキャッチコピー

2. **コンセプト**
   - 店舗のコンセプトと強み

3. **商品紹介**
   - 主な商品の紹介

4. **沿革**
   - 店舗の歴史と経緯

5. **ご予約**
   - 予約方法と注意事項

6. **SNS・ECサイト**
   - 各種SNSアカウントへのリンク

7. **フッター**
   - 店舗情報とコピーライト

## 使用方法

1. `index.html`をブラウザで開く
2. 各セクションを確認
3. 必要に応じて画像やテキストを更新

## カスタマイズ

### 画像の追加

画像プレースホルダーを実際の画像に置き換えてください：

```html
<div class="image-placeholder">
    <img src="your-image.jpg" alt="説明">
</div>
```

### 色の変更

`styles.css`の`:root`セクションでカラーパレットを変更できます：

```css
:root {
    --primary-color: #f4a5ae;
    --secondary-color: #ffd3d9;
    --accent-color: #ff6b9d;
    /* ... */
}
```

### SNSリンクの追加

`index.html`のSNSセクションでリンクを更新してください：

```html
<a href="https://instagram.com/your-account" class="sns-link instagram">
    <span>Instagram</span>
</a>
```

## 注意事項

- 実際の画像はLINEで受け取った画像に置き換えてください
- SNSアカウントのURLは実際のURLに更新してください
- 営業時間やアクセス情報は実際の情報に更新してください
- ロゴデータは実際のロゴに置き換えてください
